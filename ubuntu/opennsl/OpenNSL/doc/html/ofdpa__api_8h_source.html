189 /* group table APIs */
190 
191 /*
192  The Group Table contains one entry for each Group. The table is indexed
193  by the groupId which identifies the group entry. Data is encoded into the groupId to specify the OF-DPA
194  group entry type and information required by OF-DPA to configure the datapath.
195 
196  The groupId encoding method is:
197 
198  L2 Interface, L2 Rewrite types:
199  (MSB to LSB) 4 bits encode the Group Table Entry type |12 bits of VLAN ID | 16 bits of port identifier
200 
201  L2 Multicast, L2 Flood, L3 Multicast, and L3 Interface types:
202  (MSB to LSB) 4 bits encode the Group Table Entry type |12 bits of VLAN ID | 16 bits of index
203 
204  L3 Unicast and L3 ECMP types:
205  (MSB to LSB) 4 bits encode the Group Table Entry type | 28 bits of index
206 
207 */
208
